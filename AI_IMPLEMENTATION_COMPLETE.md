## 🚀 Getting Started

### 1. **Configure API**
Navigate to **Ace SEO > Settings** and:
- Add your OpenAI API key
- Enable "AI Content Analysis"
- Test the connection

### 2. **Start Using AI**
On any post/page editor:
- **AI Titles**: Click the sparkle button next to SEO Title
- **AI Descriptions**: Click AI button next to Meta Description  
- **AI Keywords**: Click AI button next to Focus Keyword
- **AI Analysis**: Use the AI Analysis buttons in the SEO tab

### 3. **Advanced Features**
- **Content Analysis**: Get detailed improvement suggestions
- **Topic Ideas**: Generate related content ideas
- **Real-time Recommendations**: AI suggestions appear in SEO analysis
